# Security Policy

## Reporting a Vulnerability

If you believe you have discovered a security vulnerability in this project, please **do not open a public GitHub issue**.

Instead, report the issue privately by contacting:

**Email:** support@byscuit.com

Please include:
- A clear description of the issue
- Steps to reproduce, if applicable
- Any relevant logs, screenshots, or proof of concept

We will acknowledge receipt of your report and investigate the issue in a timely manner.

---

## Supported Versions

Only the **latest released version** of this repository is supported with security updates.

Security fixes will be provided through:
- Patch releases
- Minor releases, depending on impact

---

## Scope

This repository contains a Google Tag Manager Custom Template that:
- Injects the Byscuit CMP client SDK
- Applies optional Google configuration flags

It does **not**:
- Process personal data
- Store consent data
- Handle backend services

Security issues related to the Byscuit CMP backend or hosted SDK should be reported through the appropriate Byscuit support or security channels.
