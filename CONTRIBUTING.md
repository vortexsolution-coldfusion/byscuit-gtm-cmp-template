# Contributing

Thank you for your interest in contributing to this project.

This repository provides a **Google Tag Manager Custom Template** for integrating the Byscuit Consent Management Platform (CMP). Due to the compliance-sensitive nature of consent tooling, contributions are intentionally restricted.

---

## Scope of Contributions

We may consider contributions that:
- Fix bugs or regressions
- Improve documentation or clarity
- Improve compatibility with Google Tag Manager or Consent Mode
- Improve code quality without changing behavior

We generally do **not** accept contributions that:
- Alter consent logic or decision-making
- Add tracking, analytics, or advertising functionality
- Change default behavior in ways that could affect compliance
- Introduce new external dependencies

---

## Submitting Issues

If you encounter a bug or documentation issue:
- Please open a GitHub issue with clear reproduction steps
- Include GTM version, browser, and relevant configuration details

Do **not** use GitHub issues to report security vulnerabilities.  
See `SECURITY.md` for vulnerability disclosure instructions.

---

## Pull Requests

If you would like to submit a pull request:
1. Fork the repository
2. Create a focused branch for your change
3. Ensure changes are minimal and well-documented
4. Clearly explain **why** the change is needed

All pull requests are subject to review and may be declined if they conflict with the project’s design goals or compliance requirements.

---

## Code of Conduct

Contributors are expected to engage respectfully and professionally.  
Harassment, abuse, or disruptive behavior will not be tolerated.
